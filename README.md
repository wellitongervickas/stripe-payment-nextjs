# Stripe Payment and Nextjs

## Getting Started

First, run the development server:

```bash
npm install
npm run dev
# or
yarn install
yarn dev
```

## Environment Variables

| Variable | Description | Default Value |
|--|--|--|
| SECRET_KEY | Stripe Secret Key | `` |
| PUBLIC_KEY | Stripe Public Key | `` |
| APP_URL |  | `http://localhost:3000` |
